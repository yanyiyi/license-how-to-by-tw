# Open Museum-Style License Link Pattern

## Use When

Use this pattern when a cultural heritage object page has citation information, license icons, badges, or linked URLs, but the visible license text is incomplete or the scope is unclear.

## Main Risk

Do not treat citation information as license information. Do not infer the whole page has one license when only the file or digital object has a license link.

## Check

1. Identify the exact image, file, digital object, or record text the user wants to reuse.
2. Inspect the license icon, badge, or linked URL.
3. Check whether the URL points to `creativecommons.org/licenses/`, `creativecommons.org/publicdomain/zero/`, `creativecommons.org/publicdomain/mark/`, or `rightsstatements.org/`.
4. Check whether metadata, description text, and image file have separate licenses.
5. Check whether the page links to an external original source.

## Output Rows

| 使用對象 | Attribution / 應標示對象 | CC 授權名稱 |
|---|---|---|
| **圖像／數位檔案：明確檔案名稱或主圖** | 依檔案旁引用資訊、典藏者、作者或平台要求標示 | 依授權連結 URL 正規化 |
| **後設資料／文字描述：藏品頁描述文字** | 依資料撰寫者、典藏者或來源網站標示 | 若無明確標示，填 `未明確標示` |

## Notes

If the license is inferred from the linked URL, say so briefly and preserve the URL as evidence.
