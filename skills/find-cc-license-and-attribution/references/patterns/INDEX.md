# Pattern Index

Use this index after the main workflow in `SKILL.md`. Choose the closest pattern, then use it as a checklist rather than a rule that overrides source evidence.

| Pattern | Use when | Check first | File |
|---|---|---|---|
| Google Docs Footer | A documentation page has `unless otherwise noted`, and code samples or trademarks may use separate terms | text vs code vs trademarks | `google-docs-footer.md` |
| Open Museum License Link | A cultural heritage page shows license mainly through icons, badges, or linked URLs | file-level license link and scope | `openmuseum-license-link.md` |
| TCMB Split License | A cultural memory or archive record separates digital object, description text, and metadata | image/object vs text/metadata | `tcmb-split-license.md` |
| OCF Explicit Attribution | A page or project gives exact attribution wording or requested credit text | required attribution sentence | `ocf-explicit-attribution.md` |

If no pattern fits, use the main workflow only.
